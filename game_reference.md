# 《像素社区》游戏设计文档（Game Reference）

## 一、游戏定位

《像素社区》是一款2D像素风、多人联网、浏览器可玩的轻生活模拟游戏，强调社交展示与长期成长循环。

核心关键词：

* 像素风
* 多人实时在线
* 轻度挂机成长
* 强社交 + 强展示
* 浏览器运行（Vue + Node.js）

---

## 二、核心玩法循环

1. 登录游戏
2. 查看角色状态
3. 处理基础生活需求（精力/心情等）
4. 选择行为（学习 / 打工 / 资源采集）
5. 社交互动（聊天 / 访问 / 点赞）
6. 获取资源与成长
7. 形成下一轮目标

---

## 三、角色系统

### 1. 等级系统

* 初始等级：1
* 最大等级：100
* 经验曲线：指数增长
* 升级规则：

  * 达到经验阈值自动升级
  * 经验值不清零（累计）

---

### 2. 生活属性

| 属性 | 说明          |
| -- | ----------- |
| 精力 | 行为消耗资源，自动恢复 |
| 心情 | 影响收益倍率      |
| 饥饿 | 影响心情与健康     |
| 健康 | 状态系统影响整体表现  |
| 清洁 | 同饥饿         |
| 武力 | 行为前置条件      |
| 智力 | 行为前置条件      |
| 魅力 | 行为前置条件      |

#### 心情收益倍率

* 极差：80%
* 差：90%
* 普通：100%
* 好：110%
* 极好：120%

---

### 3. 战斗属性

| 属性  | 说明              |
| --- | --------------- |
| 攻击  | 伤害输出            |
| 防御  | 减伤              |
| 速度  | 决定出手顺序          |
| 闪避  | 概率规避伤害          |
| 生命值 | 基于等级（白值 + 额外加成） |

---

## 四、技能系统

### 分类

* 生活技能
* 战斗技能

### 规则

* 每个技能等级：1~10
* 技能数量：20+（可扩展）
* 类型：

  * 伤害
  * Buff
  * Debuff
  * 控制

---

## 五、行为系统（挂机核心）

### 通用规则

* 支持离线挂机结算
* 所有产出走配置表

---

### 行为分类

#### 1. 教育系统

* 学历阶段：

  * 学前班 → 小学 → 中学 → 大学 → 研究生 → 博士
* 条件：

  * 金币
  * 属性要求
* 产出：

  * 属性
  * 技能
* 支持动态新增课程

---

#### 2. 打工系统

* 条件：属性 + 学历
* 产出：金币

---

#### 3. 采集系统

* 挖矿 / 伐木 / 钓鱼
* 产出：资源

---

#### 4. 商店系统

* 多商店配置
* 道具买卖

---

## 六、货币系统

| 类型  | 来源      |
| --- | ------- |
| 金币  | 打工      |
| 点券  | 充值 / 活动 |
| 成就点 | PK      |

---

## 七、PK系统

### 规则

* 自动战斗（无操作）
* 按速度排序
* 30秒内返回结果 + 战报

### 特点

* 使用实时玩家数据镜像
* 技能自动释放
* 支持装备系统

---

## 八、社交系统

### 功能

* 好友系统
* 聊天系统（全服 / 房间 / 私聊）
* 访问房间
* 点赞 / 留言板
* 排行榜

---

## 九、展示系统

### 内容

* 房间布置（家具系统）
* 角色外观（装扮系统）
* 坐骑系统
* 成就展示

---

## 十、地图系统

### 结构

* 房间制（传送点连接）
* 类型：

  * 广场（20人 + 分线）
  * 家（私有）

### 特性

* 支持实例化
* 地图可配置

---

## 十一、实时同步

同步内容：

* 玩家位置（x,y）
* 状态（坐 / 站 / 行走）

特点：

* 格子移动
* 客户端预测 + 服务端校正

---

## 十二、美术规范

* 风格：2D像素
* 视角：俯视/轻斜
* 尺寸：32x32

---

## 十三、配置系统

* 所有玩法走 JSON 配置
* Excel → JSON
* 支持热更新

---

## 十四、后台系统

* 配置编辑
* 地图编辑器
* 玩家管理
* 服务控制
