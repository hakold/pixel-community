# 像素社区 — 挂机数值与升级数值规划方案

> 基于常见线上挂机页游模式，针对当前项目数值体系进行重构规划。

---

## 一、当前数值问题诊断

### 1.1 经验曲线过陡

当前公式：`floor(100 × 1.5^(level-1))`

| 等级 | 累计经验 | 问题 |
|------|---------|------|
| 10 | 10,981 | 正常 |
| 20 | 648,879 | 开始陡峭 |
| 30 | 37,000,000 | 断崖 |
| 50 | 1,200,000,000,000 | 天文数字 |
| 100 | 7.9×10²² | 不可达 |

**问题**：增长因子 1.5 在指数曲线中过于激进。Lv20 之后升级所需时间呈指数爆炸，玩家在中期就会卡住。

### 1.2 金币循环断裂

- 学习消耗金币（100 → 150,000），打工产出金币（20 → 10,000）
- 用基础工作赚够 PhD 的 150,000 金币需要跑 7,500 次清洁工
- 同级打工收入无法覆盖同级学习支出（如：大学需 15,000 金币，同级白领每 30 分钟赚 800）

### 1.3 精力经济不合理

- 消耗范围 5-50，回复 1/分钟 = 100 分钟回满
- 5 分钟河钓消耗 5 精力 → 5 分钟回满，无意义
- 16 小时 PhD 消耗 50 精力 → 50 分钟回满，精力不是限制因素
- 属性衰减（0.2/min）在长任务中被回复覆盖，衰减形同虚设

### 1.4 缺少消耗出口

- 背包有 18 种物品，全部只能堆叠，无法使用/出售/合成
- 生活属性（饱腹/清洁/健康）只衰减不恢复
- 战斗属性已定义但无战斗玩法
- `coupon`、`achievementPoint` 存在于数据库但无处产出也无处消耗

## 二、目标规划原则

参考成熟的线上挂机页游（摩尔庄园页游、奥比岛、Travian、Melvor Idle 等），制定以下设计原则：

| 原则 | 说明 |
|------|------|
| **分段节奏** | 前期快节奏（分钟级反馈），中期稳定推进（小时级），后期长线养成（天级） |
| **多重门禁** | 等级、属性、金币、学历四重门禁交错，避免单一瓶颈 |
| **收支平衡** | 每个阶段的打工收入能覆盖该阶段的学习支出，采集产出有使用场景 |
| **精力为锚** | 精力是唯一的时间货币，所有行为价值和消耗以精力为基准对齐 |
| **衰减驱动** | 生活属性衰减 + 恢复手段 = 持续的维护需求，制造经济循环 |
| **渐进解锁** | 每 5-10 级有可感知的新内容解锁，避免长期无变化 |

## 三、经验曲线重构

### 方案：分段多项式

采用 `base × level^exponent` 的分段曲线，在不同等级段使用不同指数，兼顾前期爽感和后期耐玩性。

```
Lv  1-20:  exp = floor(80 × level^2.0)      前期——快速成长
Lv 21-40:  exp = floor(120 × level^2.3)     中期——稳定推进
Lv 41-60:  exp = floor(200 × level^2.5)     中后期——策略规划
Lv 61-80:  exp = floor(500 × level^2.6)     后期——长线养成
Lv 81-100: exp = floor(1000 × level^2.8)    大后期——终极目标
```

### 对比旧曲线

| 等级 | 旧曲线经验 | 新曲线经验 | 差异 |
|------|-----------|-----------|------|
| 10 | 10,981 | 8,000 | 相近 |
| 20 | 648,879 | 32,000 | **降低 20 倍** |
| 30 | 37,000,000 | 92,500 | **降低 400 倍** |
| 50 | 1.2×10¹² | 470,000 | **降低 260 万倍** |

新曲线使 Lv100 成为一个约 300 天可达的长期目标，而非纯数学不可能。

## 四、升级属性成长

### 4.1 升级奖励（每级）

| 属性 | 当前 | 建议 |
|------|------|------|
| 最大 HP | +5 | +3 |
| 攻击 | +2 | +1 |
| 防御 | +2 | +1 |
| 速度 | +1 | +0.5 (每 2 级 +1) |

当前数值过于膨胀（Lv100 = 595 攻击，远超需要）。降低让战斗数值更可控。

### 4.2 生活属性解锁

生活属性不是纯数值堆砌，而是门禁条件：

| 等级区间 | 力量上限 | 智力上限 | 魅力上限 |
|----------|---------|---------|---------|
| 1-10 | 20 | 20 | 20 |
| 11-20 | 40 | 40 | 40 |
| 21-30 | 60 | 60 | 60 |
| 31-50 | 80 | 80 | 80 |
| 51-100 | 100 | 100 | 100 |

生活属性主要通过**行为奖励**和**物品使用**提升，而非升级自动增长。

## 五、精力经济重构

### 5.1 核心定位

精力 = 时间货币。所有行为的价值以"每点精力产出"为基准对齐。

### 5.2 回复速率

| 当前 | 建议 | 理由 |
|------|------|------|
| 5/5min = 1/min | **3/5min = 0.6/min** | 100 精力需约 2.8 小时回满，制造有意义的等待 |

### 5.3 消耗标准化

| 行为级别 | 时长 | 精力消耗 | 回满可执行次数 |
|----------|------|---------|--------------|
| 初级 (Lv1-10) | 2-10 分钟 | 3-8 | 12-33 次 |
| 中级 (Lv11-25) | 10-30 分钟 | 8-15 | 6-12 次 |
| 高级 (Lv26-50) | 30-120 分钟 | 15-25 | 4-6 次 |
| 精英 (Lv51+) | 2-8 小时 | 25-40 | 2-4 次 |

### 5.4 精力效率表（每点精力产出）

以初级行为为基准 1.0：

| 行为 | 每精力经验 | 每精力金币 | 特点 |
|------|-----------|-----------|------|
| 学习 | 高 (2-4) | 无 | 解锁内容、提升属性 |
| 打工 | 低 (0.5-1) | 高 (2-8) | 金币主要来源 |
| 采集 | 中 (1-2) | 中 (物品价值) | 材料产出 |

## 六、金币经济重构

### 6.1 来源与消耗

```
金币来源: 打工行为 → 每天约 200-2000 (随等级增长)
金币消耗: 学习费用 + 道具购买 + 装备购买(未来)
```

### 6.2 学习-打工平衡表

同级打工应当在 3-5 次内赚够同级学习费用：

| 阶段 | 学习费用 | 对应打工收入/次 | 打工次数 | 耗时 |
|------|---------|--------------|---------|------|
| 学前 | 100 | 30 | 4 | 20 分 |
| 小学 | 500 | 80 | 7 | 70 分 |
| 初中 | 2,000 | 200 | 10 | 150 分 |
| 高中 | 5,000 | 500 | 10 | 250 分 |
| 大学 | 12,000 | 1,000 | 12 | 360 分 |
| 研究生 | 30,000 | 2,500 | 12 | 360 分 |
| 博士 | 80,000 | 5,000 | 16 | 480 分 |

> 原则：越高级的学习需要更多次数的高级打工来支撑。玩家在每阶段有 2-5 天的金币积累期。

### 6.3 建议新增的打工档次

当前打工最高收入为 10,000（首席科学家），但博士费用为 150,000。建议调整：

| 打工 | 当前金币 | 建议金币 | 原因 |
|------|---------|---------|------|
| 清洁工 | 20 | 25 | 微调 |
| 咖啡店员 | 35 | 50 | |
| 外卖骑手 | 40 | 70 | |
| 勤工俭学 | 80 | 120 | |
| 家教 | 180 | 300 | |
| 企业实习 | 400 | 600 | |
| 白领职员 | 800 | 1,200 | |
| 研究员 | 1,200 | 2,000 | |
| 科学家 | 3,000 | 4,000 | |
| 教授 | 6,000 | 8,000 | |
| 首席科学家 | 10,000 | 15,000 | |

## 七、属性衰减与恢复循环

### 7.1 生活属性全生命周期

```
         ┌──────────────────────────────┐
         │         正常状态               │
         │  (属性 60-100，心情倍率 1.0+)  │
         └──────────┬───────────────────┘
                    │ 执行行为 → 时间流逝 → 属性衰减
                    ▼
         ┌──────────────────────────────┐
         │         衰减状态               │
         │  (属性 20-60，心情倍率 0.9)    │
         └──────────┬───────────────────┘
                    │ 使用恢复道具/执行恢复行为
                    ▼
         ┌──────────────────────────────┐
         │         恢复循环               │
         │  吃饭→饱腹↑  休息→精力↑       │
         │  洗澡→清洁↑  娱乐→心情↑       │
         └──────────────────────────────┘
```

### 7.2 衰减率调整

| 属性 | 当前/分钟 | 建议/分钟 | 满值耗尽时间 |
|------|----------|----------|------------|
| 精力 | 0.2 | 0.2 | 8.3 小时 |
| 饱腹 | 0.15 | **0.12** | 13.9 小时 |
| 清洁 | 0.1 | **0.08** | 20.8 小时 |
| 心情 | 0.05 | **0.04** | 41.7 小时 |
| 健康 | 0.3(饥饿时) | **0.15**(饥饿时) | — |

> 降低衰减率使玩家不需要过于频繁地维护状态。心情和清洁设计为每天关注 1-2 次即可。
> 精力有回复，净效果为正（+0.6 回复 - 0.2 衰减 = +0.4/min 净增）。

### 7.3 恢复手段规划（后续实现）

| 属性 | 恢复方式 | 恢复量 | 冷却/消耗 |
|------|---------|--------|----------|
| 精力 | 自动回复 | 3/5min | 全局 |
| 精力 | 食物 | +20~50 | 消耗金币/材料 |
| 饱腹 | 食物 | +30~80 | 消耗金币/材料 |
| 清洁 | 洗澡(城镇设施) | +40~60 | 消耗金币 |
| 心情 | 娱乐(城镇设施) | +20~40 | 消耗金币 |
| 健康 | 医疗(城镇设施) | +30~50 | 消耗金币 |

## 八、采集产出与使用场景

### 8.1 采集品价值体系

| 等级 | 材料 | 单价(金币) | 用途 |
|------|------|-----------|------|
| Lv1 | 铜矿石、松木、鲤鱼 | 5-10 | 初级合成/任务 |
| Lv5-10 | 铁矿石、橡木、鲈鱼 | 15-30 | 装备强化/出售 |
| Lv15-20 | 金矿石、黑檀木、金枪鱼 | 50-100 | 高级合成/出售 |
| Lv25-30 | 钻石原石、远古木、鱼翅 | 200-500 | 稀有道具 |
| Lv30+ | 秘银矿、珍珠、古钱币 | 500-2000 | 顶级道具/收藏 |

### 8.2 材料经济循环

```
采集行为 → 材料掉落
           ├── 直接出售给 NPC → 金币（保底收入）
           ├── 合成消耗品 → 维持生活属性
           ├── 合成装备 → 提升战斗属性
           └── 任务提交 → 经验/声望/成就
```

## 九、学习-打工-采集三角关系

```
              学习 (study)
              /          \
             /            \
        解锁工作          提升生活属性
           /                \
          /                  \
    打工 (work) ───赚钱──→ 采集工具/许可证
          \                  /
           \                /
            金币 ←── 出售采集品
                       ↓
                  采集 (mining/woodcut/fishing)
```

**设计意图**：
- 学习解锁高级工作 → 高级工作赚更多金币
- 学习提升属性 → 属性满足高级采集门槛
- 采集产出材料 → 出售换金币 → 金币投入学习
- 三者互相依赖，形成闭环

| 场景 | 验证标准 |
