# Generated Config

这里是运行时消费的 JSON 配置目录。

原则：

- 不直接手改这里的受管文件
- 受管文件应由导表脚本生成

- `game.json`
- `study.json`
- `work.json`
- `mining.json`
- `woodcut.json`
- `fishing.json`
- `skills.json`
- `items.json`
- `shops.json`
- `recoveryActions.json`
- `levelExp.json`

- `maps/*.json`
