# AI Workspace

这个目录只服务于项目协作与 AI 接力，目标是让后续任何一个 AI 或开发者都能快速知道：

- 已经确认了什么
- 目前做到哪里
- 下一步该做什么

## 文件约定

- `STATUS.md`
  当前状态总览，必须包含”已做内容 / 当前内容 / 未来规划”。
- `ROADMAP.md`
  阶段目标、里程碑、验收标准。
- `DECISIONS.md`
  已确认的架构约束、重要取舍、暂不执行的事项。
- `ART_ASSETS.md`
  AI 美术素材协作指南。素材清单、提示词模板、生成流程、注册步骤。其他 AI 模型接手素材生成时先读这个。
- `M1_CHECKLIST.md`
  M1 配置基线回归检查清单。
- `M2_BLUEPRINT_ACCEPTANCE.md`
  M2 蓝图主城验收文档。记录选中的主城蓝图、POI 映射、当前可玩链路、剩余调优项。
- `SESSION_SUMMARY.md`
  每次完成阶段性工作后的会话总结。新 AI 接手时先读这个，再读 STATUS.md。

## 更新规则

- 每次进入新的阶段前，先更新 `STATUS.md`
- 每次确认新的方向或约束，补充到 `DECISIONS.md`
- 每次完成里程碑或拆分任务，更新 `ROADMAP.md`

## 当前共识

- Excel 作为策划填写源
- JSON 作为运行时配置
- 地图格子继续由地图编辑器维护，不放进 Excel
- 配置目录从前后端代码目录中隔离，统一迁移到仓库根目录独立维护
